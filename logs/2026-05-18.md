# 2026-05-18

## Создан Vite + React демо-сайт из дизайна Claude Design

Конвертирован проект med-app из standalone HTML/Babel в полноценный Vite + React проект.

### Что сделано
- Создан `package.json`, `vite.config.js` (base: './'), `index.html` с подключением шрифта Golos Text
- `src/Shared.jsx` — design tokens (RUSSDENT_COLORS, RUSSDENT_GRADIENT, FF) + все базовые компоненты (RCard, RBtn и др.)
- `src/IOSFrame.jsx` — iOS-обёртка (Dynamic Island, Status Bar, Home Indicator)
- `src/Navigation.jsx` — нижнее меню с SVG-иконками
- `src/screens/` — 5 экранов: Home, Booking, Results, History, Profile
- `src/App.jsx` + `src/main.jsx` — точка входа с адаптивным масштабированием под размер окна
- Сборка прошла успешно (181 KB JS, 632ms)

### Деплой на GitHub Pages
## Адаптив и SCSS-модули

- `src/App.module.scss` — SCSS-модуль для лейаута: десктоп (iOS-фрейм) и мобайл (полноэкранный)
- `src/Navigation.module.scss` — `padding-bottom: max(30px, env(safe-area-inset-bottom))` для Home Indicator
- CSS-переменная `--screen-top-pad` на `.mobileApp` — экраны читают её вместо `68px`; на десктопе fallback `68px`
- `viewport-fit=cover` в `index.html` — включает поддержку Safe Area на реальных iPhone
- `useIsMobile()` хук в App.jsx — `matchMedia(max-width: 600px)`, рендерит мобайл/десктоп ветку
